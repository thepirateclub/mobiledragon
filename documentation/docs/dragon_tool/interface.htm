Dragon Tool User Manual  

User Interface

DragonTool window consists of a few elements:

Menubar

File

Here you can load, save project and export it to a dragon tool format.

Edit

In this menu following operations are located:

  • Standart operations: Undo/Redo.
  • Creation/Removing operations: Delete, Copy, Instance. For a difference between copies and instances see Basic concepts.
  • Selection operations: Select Children, Select Parents, Select None.
  • Operations which deal with nodes order and their position in tree hierarchy.

    Move To Back, Move To Front - makes selected object first (or last) child of it's parent.

    Swap - swaps two selected nodes.

    Group - creates a new transform node and attaches all selected objects to it.

    Link - Two nodes must be selected to perform this operation. It creates an Instance of the first selected object and parents it to the second object.

    Parent - attaches all selected objects to the last selected object.

    Unparent - detaches selected objects from their parents.

  • Transform operations (can be performed only on the Transform and Animation objects): Rotate 90, Rotate -90, Flip Horizontal.
  • Convertion operations. You can convert some of the objects into another objects. (For example convert Transform into Animation).
  • Add Custom Property. Here you can add a custom property to selected objects and then export it into Mobile Dragon resource file. A dialog appears where a name of the property and it's type should be defined. More info about property types see at Scene objects.
Sprite
  • Render - renders currently selected objects and writes it's images into the output folder of the project.
  • Import - select images to import them into the project. When image on the disk is changed it automaticaly updates in the scene.
Window
  • Brushes - displays all the children of the selected node as a brushes. Then this objects can be painted on the Live Objects.
Create
    Here is a list of the objects that can be created. For creation of a Sprite use "Sprite -> Import" command.

Sprite Set

  • Pack Row, Pack Column - this commands arrange children of selected node into row or column.
  • Sort by Name - sort children of selected node by name.

Animation

  • Add Frame - add frame to selected animation
  • Add Layer - add one more layer to selected animation
  • Render Animation - creates png files of animation frames and puts them into output dir of the project.

Edit View

This is a main work area. Here you can see and overview of the project or view only one of the objects in the scene. When the project is loaded it is shown in this window. Then you can right click on the objects in the Outliner panel to view them

Property Editor

When you select an object in the scene a Property Editor displays all the properties of this object (including custom properties). Click on the property value to edit it. There are read-only properties which cannot be set. They are shown on the gray background. If property value is shown on a red background that means that it depends on another property. (For example 'frameNum' property of an Animation object depends on 'frameNum' property of Time object).

Outliner Panel

In this window an hierarchy of the scene nodes is displayed. You can expand/collapse this nodes to show/hide their children. Drag and drop a node onto another one to attach it and right click them to view in the edit view.

Logger Panel

This is a place where program messages are displayed (info, warngins or error messages).

Toolbar

Here is a current tool selected. Hotkey for each button can be seen in tooltips.
First two tools are Select tool and Move tool. They work with objects in the scene. Only thouse objects that have 'translateX' and 'translateY' properties can be moved. Hold alt while movind object to create it's copy. Hold ctrl to add to selection and alt to substract from selection.
Pan and Zoom tools work like their Photoshop analogues. To zoom in just left click on edit view. To zoom out hold alt and left click.
Make live. Press it to make selected object 'live object'. You can paint with brushes on live objects.
Buttons Select on Tile Map, Move on Tile Map and Eraser work with Tile Maps.

Tool Info Panel

Every time you select a tool some info concerning it displayed here.

Timeline and Player Panel

This two elements are interactiong with the Time node. Click (or click and drag) on the Timeline to set a time value. 'Rewind' button sets Time node frameNum property at 0. Also there is a button that brings an "FPS" dialog where you can set animation speed.
