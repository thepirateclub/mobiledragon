MobileDragon Packdir File Creation Utiluty

The Packdir command line utility is used to compress game resource files.

It's syntax is simple:
packdir <DirName>

This will compress files from "DirName" directory including subfolders into two files packdir.dat and packdir.fat.
This files are to be deployed on Symbian or Windows Mobile device into the folder where the game executable resides and PackDir class is to be used to access it's content.

With Palm devices there is one more thing to do with those files - convert to Palm .PDB database format. Two other utilites handle this stuff:

The PackPDB command line utility splits file into 60K volumes and prepares source file for PDBC( Palm DataBase Compiler ) utility. Obtain it from http://obermuhlner.com/public/Projects/Palm/PDBC/

Usage is:
PackPDB <File> <CreatorID>

Where "File" is filename to split, "CreatorID" is your Palm application CREATOR ID.

Use this like:
PackPDB packdir.dat MBDR

This will split file packdir.dat into files volume*.bin and create file "result.pdbc" - this is source file for PDBC utility.

Then launch PDBC utility like this
PDBC result.pdbc

This will produce file "out.pdb". Rename it to packdirdat.pdb.
ren out.pdb packdir.pdb

Then repeat it with packdir.fat file:
PackPDB packdir.fat MBDR
PDBC result.pdbc
ren out.pdb packdirfat.pdb

This two files can be deployed on Palm device with your game prc file.
