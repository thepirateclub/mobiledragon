MobileDragon Guide

Description:

The MobileDragon is a cross-platform high performance realtime 2D & 3D game engine written in C++. It comes with documentation and samples and integrates features for visual representation like dynamic lighting, LOD system, particle systems, character animation, and collision detection. All this is accessible through an easy to use C++ interface.

Main features:

  • Software 3D render (z – buffer, Gouraud lightings, perspective correction, alpha-blending).
  • Software 2D render.
  • Platform - independent. Runs on Windows Mobile 2003 Pocket, Windows Mobile 2003 Smartphone, Symbian OS 6, 7, 8, Palm Os 5.x.
  • Supports following compilers: eVC 4 SP 4, Visual Studio 6 SP 6, Visual Studio 2005, Metrowerks Codewarrior 10 (Palm), gcc 2.9.6.
  • 3DStudio Max model exporter plugin.
  • Morphing – animation of human-like characters.
  • Joint animation.
  • Particle systems.
  • 2d & 3d sprites.
  • Portal system.
  • LOD management.
  • Template container library, math.
  • 2d & 3d font.
  • Resource storage in compressed container.
  • Collision detection system.
  • Network support: Bluetooth.
  • Sound support: wav & mod files playback.
  • Input system supports device keyboard & stylus.
