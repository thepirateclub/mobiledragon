Dragon Tool User Manual  

Program overview

Dragon Tool is a program for collecting and editing resources of the game so they can be loaded into the Mobile Dragon engine. Dragon Tool can be used as:

Map editor

It allows you to create a tile maps with any cell configuration and any level of complexity. You can create isometrix maps, maps with hexagonal cells or even define any shape of a cell. Also there are so called 'Indexed' tile maps (it's like indexed bitmaps which have a palette). You can define one or more 'palettes' of tiles for the one tile map and and then switch between them.

Animation editor

Import sprites, place them in separate frames, tune they placement, define delay - and animation is ready. It's ofen comes in handy to divide animations of a characters in a few layers (weapon layer, legs layer, body layer and so on). With Dragon Tool you can create animations with any number of layers.

Iterface editor

It's always comfortable to define and modify such things as a "health bar location" or a "button location" in the editor, but not in the game code.

Dragon Tool is always improving and we are looking forward to make the process of game development with this program more comfortable and interesting.
