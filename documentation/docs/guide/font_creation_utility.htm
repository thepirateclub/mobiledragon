MobileDragon Texture Font Creation Utility

The TextureFont utility is used to produce MobileDragon 2d and 3d fonts binaries from system installed fonts.

Launch it, choose font family, style & size, and additional spacing between characters.

Two files will be created in program directory - <FontName>.bin and <FontName>.pcx - font picture and binary. Keep those files together in resource directory, and use <FontName> for initializing Font2D or Font3D class objects.
