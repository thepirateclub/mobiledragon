Working with Animations

This small tutorial shows how to create and modify animations. Often it's better to make animations using layers, 'cos this helps to save memory.
Download a project, unpack it and load start.xml.
Import all sprites "Sprite -> Import" and create an Animation object: "Create -> Animation". We will have one animation object with one frame attached to it and one layer attached to this frame. Animation object consist of frame objects and it displays only one of them at a certan time moment.
Then we should combine this sprites to create animation frames. Firstly we create draft of animation (only layers and body) and then attach some effects and shadow.
Select samuraj_x15_legsL1 and samuraj_x15_swordL1 and parent them to a frame0. Right click frame0 in the Outliner panel and arrange this layers according to a screenshot.

After that copy frame0, remove a body layer from it and attach samuraj_x15_swordL2 to this frame. Again, right click on this frame and arrange layers.

Repeat this steps (copy frames, delete and attach layers, arrange them) to make the following sequence:

Then we add some effects to animation. Add shadow to every frame and arrange it and add two effects to ephase some frames.

Right click on animation and press 'play' button. It would be better to add little pause after stab. Just select the frame where the sword has stopped and change the delay property of this Frame object to 4.
You'll probably wish to tune animation and add some extra frames. To change the order of frames use "Edit -> Swap", "Edit -> Move to Front" and "Edit -> Move to Back" commands.

To move all the layers you can select animation, right click on it in the outliner panel and choose command "Edit -> Select Children" twice. After that all layers of this animation will be selected and you can move them to the origin.
