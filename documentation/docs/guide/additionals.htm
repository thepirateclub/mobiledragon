MobileDragon Additional Components

There are the additional files residing in the "MobileDragon\extras" folder, which need for MobileDragon based application.

1. Folder "MobileDragon\extras\WindowsMobile" contains following files:

mdbt_ms.dll, mdbt_wdk.dll - MobileDragon bluetooth support dll's for Microsoft Bluetooth stack;
This two files must be deployed into the same directory as MobileDragon Bluetooth enabled game. If your application does't use Bluetooth, these files are not required.

BtCoreIf.dll, BtSdkCE30.dll - Broadcom dll's for Widcomm stack support;
This two files must be deployed into "Windows" directory on the Windows Mobile device. Also it's necessary to mark these files with "shared" attribute. If this folder already contains files with same names, don't replace them.

2. Folder "MobileDragon\extras\SymbianS60" contains following files:

hss.dll - Sound support dll for Symbian S60.
This file must be deployed into the same directory as any MobileDragon based game.

3. Folder "MobileDragon\extras\SymbianUIQ" contains following files:

hss.dll - Sound support dll for Symbian UIQ 2.1.
This file must be deployed into the same directory as any MobileDragon based game.
